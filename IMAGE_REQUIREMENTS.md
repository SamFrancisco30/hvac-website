# Image Requirements — ArcticPrecision HVAC

All images should be saved as `.jpg` at **1600 × 900 px minimum** (16:9), optimised for web.
Place each file at the path shown. The site uses them as hero backgrounds — they're overlaid with a dark gradient, so a slightly darker or moodier exposure works better than bright studio shots.

## Notes

- All images are overlaid with a **dark blue gradient** (`rgba(0,33,71)`) and displayed at **20–30% opacity** as hero backgrounds. Images don't need to be perfectly composed — texture and mood matter more than detail.
- **No stock-photo clichés**: avoid obvious posed smiles, fluorescent-lit offices, or cartoon-style illustrations.
- If using AI image generation (Midjourney, DALL·E 3, Stable Diffusion): append `--ar 16:9 --v 6` (Midjourney) or specify `aspect ratio 16:9, photorealistic` for other tools.
- Minimum file size after optimisation: aim for **under 300 KB** per image using tools like Squoosh or ImageOptim.
